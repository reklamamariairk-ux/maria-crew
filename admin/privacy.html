Политика конфиденциальности

Дата вступления в силу: 12 мая 2026 · Контакт: bbf2018.ru@gmail.com

Maria Crew — внутреннее приложение мотивации сотрудников кондитерских «Мария» (Иркутск). Эта политика описывает, какие данные мы собираем, зачем и как ими распоряжаемся.

1. Какие данные собираем

  • ФИО — для идентификации сотрудника
  • Email (опц.) — для входа в мобильное приложение и обратной связи
  • Телефон — для связи и интеграции с 1С (начисление премий)
  • Telegram ID и username (опц.) — для уведомлений через Telegram-бот
  • Фото профиля (опц.) — берётся из Telegram-аккаунта
  • Точка работы — конкретная кондитерская «Мария»
  • История активности — чек-ины, ответы в квизе, начисления и списания Mария-монет, полученные карточки, заявки на призы из Maria Store
  • Push-токены (FCM / APNs) — только для отправки уведомлений в мобильном приложении. Никому не передаются и хранятся только локально на нашем сервере.

2. Как используем

  • Вход и идентификация — email + одноразовый PIN, либо вход через Telegram Mini App. Постоянные пароли для рядовых сотрудников не используются.
  • Уведомления — Telegram-бот и push в мобильном приложении.
  • Внутренние отчёты — рейтинги сотрудников и точек, аналитика квиза, топ-исполнителей за месяц.
  • Бухгалтерия — синхронизация с 1С для начисления премий.

3. Где храним

  • База данных PostgreSQL — основной хост: Render (регион Oregon, US). Резервная база в Neon (EU) на период миграции (до 1 июня 2026).
  • Бэкенд — Render Web Service (Oregon, US).
  • Файлы (фото героев) — Cloudinary (без личных данных).
  • Push-токены — в нашей PostgreSQL, исходящая отправка через Firebase Cloud Messaging.

Доступ к данным имеют только администраторы организации «Мария». AI-сервисы (Anthropic, OpenAI и другие) к личным данным сотрудников не имеют доступа.

4. Срок хранения

  • Активный сотрудник — пока работает плюс 6 месяцев после увольнения для отчётности.
  • Удалённый аккаунт — личные данные (имя, телефон, email, Telegram) удаляются немедленно. История транзакций по Mария-монетам обезличивается, но сохраняется для бухгалтерии.

5. Ваши права

  • Удалить аккаунт — кнопка в настройках мобильного приложения. Удаляется немедленно.
  • Запросить копию данных — напишите на bbf2018.ru@gmail.com.
  • Исправить данные — обратитесь к руководителю точки или на тот же email.
  • Отозвать согласие — равнозначно удалению аккаунта.

6. Третьи стороны

  • Telegram (политика: telegram.org/privacy) — авторизация через Mini App и уведомления через бот.
  • Firebase Cloud Messaging (политика: firebase.google.com/support/privacy) — push в мобильном приложении.
  • Cloudinary (политика: cloudinary.com/privacy) — хранение фото героев. Личных данных сотрудников там нет.
  • Render, Neon — хостинг бэкенда и базы данных.

7. Безопасность

  • Постоянных паролей для рядовых сотрудников нет — только одноразовые PIN-коды, действующие 10 минут.
  • Пароли администраторов хранятся в формате scrypt-хеша с уникальной солью; минимальная длина — 8 символов, проверяется чёрный список распространённых паролей.
  • JWT-токены подписаны HMAC-SHA256, действуют 30 дней (мобильное приложение) или 7 дней (админ-панель).
  • Все сетевые запросы — HTTPS. Соединение с базой данных — TLS-only.
  • Rate-limit на эндпоинты входа: 10 попыток за 15 минут с одного IP.

8. Дети

Приложение не предназначено для лиц младше 18 лет.

9. Изменения

Об изменениях политики предупредим через мобильное приложение и Telegram-бот. История версий ведётся в Git-репозитории проекта.
