# Maria Crew — чек-лист тестирования

> Привет! Спасибо что согласился потестировать приложение **Maria Crew** перед запуском.
> Это твоя задача — пройти все пункты и **сразу написать руководителю** если что-то непонятно или не работает.
> Время на всё ~15 минут.

## Перед началом

- ✅ Скачай APK на свой Android-телефон (ссылку даст руководитель)
- ✅ Или открой приложение в Telegram через **@Mariaprod_bot** → кнопка «🚀 Открыть Maria Crew»
- ⚠️ Если что-то непонятно — НЕ пытайся «разобраться сам», сразу пиши руководителю. Это и есть смысл теста — найти неудобства.

---

## Часть 1 — Регистрация и вход (5 минут)

### 1.1. Первый запуск
- [ ] Открой приложение. Какая иконка на рабочем столе телефона? (девушка на красном фоне?)
- [ ] При запуске показывается логотип на красном (splash screen)?
- [ ] Через сколько секунд появляется экран ввода телефона / email?

### 1.2. Вход по телефону
- [ ] На login screen введи свой телефон (тот что записан у руководителя)
- [ ] Нажми **«Получить код»**
- [ ] **Куда пришёл код?** (Telegram, Email, оба, никуда?)
- [ ] Введи код → ты в приложении?

### 1.3. Если код не пришёл
- [ ] Попробуй переключить на **«✉️ По email»**
- [ ] Введи свой email (тот что в профиле)
- [ ] Получил код? Куда?

### 1.4. Если телефона нет в системе
- [ ] Должна появиться форма регистрации (имя + точка + email)
- [ ] Зарегистрируйся, получи код, войди

---

## Часть 2 — Профиль и настройки (3 минуты)

### 2.1. Открой свой профиль
- [ ] **Тапни на аватар** в шапке (кружок с буквой имени, слева сверху)
- [ ] Открылась форма? Какие поля видны?

### 2.2. Заполни всё что не заполнено
- [ ] Имя: должно быть твоё реальное имя
- [ ] Телефон: твой реальный номер (это важно — по нему начисляются награды)
- [ ] Email: твой email
- [ ] Аватарка: загрузи фото (нажми 📷)
- [ ] Сохрани

### 2.3. Колокольчик 🔔
- [ ] Справа от огонька серии — иконка колокольчика
- [ ] Тапни → открывается список уведомлений?
- [ ] Если есть — закрой шторку

---

## Часть 3 — Основные функции (5 минут)

### 3.1. Карточки коллекции
- [ ] Открой вкладку **«Карточки»** внизу
- [ ] Видишь сетку из 12 героев?
- [ ] Тапни на любого героя → открывается описание?
- [ ] Закрой описание

### 3.2. Монеты
- [ ] Вкладка **«Монеты»**
- [ ] Видишь свой баланс?
- [ ] История начислений показана?

### 3.3. Квиз
- [ ] Вкладка **«Квиз»**
- [ ] Доступны вопросы? Сколько?
- [ ] Ответь на 1-2 вопроса
- [ ] За правильный ответ начисляются монеты?

### 3.4. Рейтинг
- [ ] Вкладка **«Рейтинг»**
- [ ] Видишь рейтинг своей точки?
- [ ] Видишь свою позицию?

### 3.5. Magazine / Магазин
- [ ] Вкладка **«Магазин»**
- [ ] Видишь призы? Карточки + Монеты?
- [ ] Можешь попытаться обменять (даже если не хватает)?

### 3.6. Серия дней (огонёк)
- [ ] В шапке огонёк 🔥 с цифрой
- [ ] Тапни — отметился сегодня?
- [ ] Цифра увеличилась? Или показалось «уже отмечался»?

---

## Часть 4 — Уведомления (2 минуты)

- [ ] Закрой приложение
- [ ] Попроси руководителя начислить тебе монеты (любую сумму)
- [ ] Получил уведомление в Telegram?
- [ ] Открой приложение → есть запись в колокольчике 🔔?
- [ ] Баланс монет обновился в шапке?

---

## Часть 5 — Что бесит / непонятно?

Запиши ВСЁ, даже мелочи:

1. _____________________________________________________
2. _____________________________________________________
3. _____________________________________________________
4. _____________________________________________________
5. _____________________________________________________

Примеры что считается важным:
- «Не понял что такое X»
- «Кнопка слишком маленькая»
- «Долго грузилось при первом открытии»
- «Текст плохо читается»
- «На моём телефоне (модель X, Android Y) что-то странно»
- «Когда нажимаешь Y, ничего не происходит»

## Как отправить фидбэк

**Вариант 1** — в приложении (если работает):
- Внизу есть кнопка **«Сообщить о проблеме»** — она шлёт тебе руководителю мгновенно

**Вариант 2** — в Telegram:
- Напиши руководителю в @личку с тегом `#тест-maria-crew`

**Вариант 3** — голосом:
- Просто позвони руководителю и расскажи
